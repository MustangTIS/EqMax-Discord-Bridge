==================================================
  EqMax-Discord-Bridge Project (v5)
==================================================

この度は本ツールをご利用いただきありがとうございます。
このパッケージは、強震モニタ表示ソフト「EqMax」の運用を
劇的に便利にするための統合ツール群です。

--------------------------------------------------
【同梱ツールの紹介】
--------------------------------------------------
● [00-TOP_HUB]
   全ての機能を呼び出すメインメニューです。まずはこれを実行してください。

● [01-Eq_Initialize] (初期化・最適化)
   EqMaxの面倒な初期設定（キャプチャ設定や不要機能のオフ）を
   一括で行い、動作を軽量化・安定化させます。

● [02-Eq_Discord] (Discord連携)
   EqMaxの地震速報をDiscordに転送するための設定ツールです。
   WebHook URLを登録するだけで連携が完了します。

● [03-Eq_Reset] (リセット)
   設定をやり直したい場合や、クリーンな状態に戻したい場合に使用します。

● [オプション：Cleaner]
   溜まりすぎたログやキャプチャ画像を定期清掃します。
   
 ● [オプション：Watchdog]
   EqMaxのフリーズや強制終了及びメモリリークを監視し、自動復旧させます。
   ※Discord連携アプリにはすでにこの機能が組み込まれています。
   Discord連携を使わない環境化のEqMAXを安定化させたい場合につかってください。

--------------------------------------------------
【導入ステップ（推奨手順）】
--------------------------------------------------
1. 「00-TOP_HUB」を実行します。
2. 「01. EqMax初期化」を選択し、指示に従って最適化を行います。
3. 「02. Discord連携」を選択し、お使いのDiscordサーバーの
   WebHook URLを入力して「適用」します。
4. 設定が終わったら「監視ボットの起動」を行うことで、
   Discordへの転送と死活監視がスタートします。

--------------------------------------------------
【よくある質問・注意点】
--------------------------------------------------
・EqMaxが起動していない場合は、ボットが自動的に最小化状態で起動します。
・Windowsのファイアウォール警告が出た場合は「アクセスを許可」してください。
・config.json などの設定ファイルを直接編集する必要はありません。
  (全てツール画面から設定可能です)
