==================================================
EqMax-Discord-Bridge Project (v10.0.0 Update)
==================================================

この度は本ツールをご利用いただきありがとうございます。
本パッケージは、強震モニタ表示ソフト「EqMax」の運用を最適化し、
地震速報を Discord, Slack, Matrix 等の複数プラットフォームへ即時通知するための
統合型防災通知ハブです。

【v10.0.0：Confirmed Seismic & Tsunami Warning Implementation】

[最新アップデート：v10.0.0]
● 確定震度情報・津波情報の統合：
  EEW（速報）に加え、詳細な「震源・震度報」および「津波警報・注意報」の自動通知を実装。
● ニュース速報スタイルフォーマット：
  確定報の表示順を『震度 ＞ 地域 ＞ 市区町村』の順に構成。
  テレビのニュース速報と同様の感覚で、被害の大きい地域を即座に判別可能です。
● Discord 2000文字制限対策：
  情報過多によるオーバーフローを自動検知し、適切な切り捨て処理を行うことで情報の不達を防止します。
● 総合テストユニットの強化：
  EEWだけでなく、確定報や津波情報のダミー送信テストに対応しました。
● 内部構造のモジュール化：
  EEW解析エンジンをメインソースから分離し、軽量化とメンテナンス性を向上させました。

【動作環境・スペック】
開発＆動作確認: Windows 11 / Windows Server 2025
動作環境: Windows 10/11, Windows Server 2022/2025 (x64専用)
必要ソフト: Discord, EqMax, Python (3.10-3.13推奨) 

【同梱ツールの紹介】
● [HTMLマニュアル/ヘルプ]
  アプリ内の全機能・設定方法を網羅した完全運用ガイド。
● [統合ハブツール (TOP_HUB)]
  全てのメンテナンス・設定機能を呼び出すメインメニュー。
● [通知ボット (Guardian) / Watchdog]
  メモリ監視およびフリーズ検知機能を備えた24時間監視エンジン。
● [EEW/情報強化テスト送信ユニット]
  確定報・津波情報を含めた通知テストが可能な診断ツール。
● [環境クリーンアップツール (Cleaner)]
  肥大化しやすいログやキャプチャ画像を整理します。

【導入ステップ】
1. 「EqMax-Discord-Bridge.bat」を実行し、統合ハブを起動します。
2. 画面上のボタンから「HTMLヘルプ」を参照し、各設定手順を確認します。
3. 「01. EqMax初期設定」で環境を最適化します 。
4. 「02. 各連携設定」で利用したい通知先を設定し、ボットを配置します。
5. 「テスト投稿」を実行し、正常に通知が届くことを確認してください。

【重要：トラブルシューティング】
ボットが起動しない等の場合は、配置ツールで作成された「(セーフモード)」ショートカットを実行してください。
自動的にPython環境の整合性チェックと修復が始まります。
