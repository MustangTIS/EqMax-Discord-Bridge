==================================================
  EqMax-Discord-Bridge Project (v8.6.0 Update)
==================================================

この度は本ツールをご利用いただきありがとうございます。
本パッケージは、強震モニタ表示ソフト「EqMax」の運用を最適化し、
地震速報を Discord, Slack, Matrix 等の複数プラットフォームへ即時通知するための
統合型防災通知ハブです。


--------------------------------------------------
【v8.X.X：ヘルプ完備・システム最適化Update】
--------------------------------------------------
[最新アップデート：v8.6.0]
- Discord通知ボット配置ツールに「既存のSNS/通知先設定の読み込み機能」を追加。
  Webhookやトークンの再入力・再確認の手間を大幅に削減しました。

[v8.5.0]

[v8.0.0系]
- ヘルプ・マニュアルシステムのHTML化による完全ガイド完備。
- 通知エンジン「Guardian Hub」の統合とプラットフォーム連携の最適化。


--------------------------------------------------
【動作環境・スペック】
--------------------------------------------------
開発＆動作確認: Windows 11 / Windows Server 2025
動作環境: Windows 10/11, Windows Server 2022/2025 (x64専用) ※ネイティブx64環境を推奨。
必要ソフト: Discord, Eqmax, Python (3.10-3.13推奨)


--------------------------------------------------
【同梱ツールの紹介】
--------------------------------------------------
● [HTMLマニュアル/ヘルプ]
    アプリ内の全機能・設定方法・対応表を網羅した運用ガイド。

● [統合管理ハブ]
    全ての機能を呼び出すメインメニューです。

● [Eqmax初期設定ツール (Patcher)]
    EqMaxの面倒な初期設定を一括で行い、動作を軽量化・安定化させます。

● [通知ボット (Guardian) / Watchdog]
    メモリ(Private Bytes)監視およびフリーズ検知機能を備えた高安定監視エンジン。

● [EqMAX初期化ツール]
    設定やログを含め全て削除し、導入直後の状態に戻します。

● [ログ・画像掃除ツール]
    肥大化しやすいログやキャプチャ画像を清掃します。


--------------------------------------------------
【導入ステップ】
--------------------------------------------------
1. 「EqMax-Discord-Bridge.bat」を実行し、統合ハブメニューを起動します。
2. 画面上のボタンから「HTMLヘルプ」を参照し、各設定手順を確認します。
3. 「01. EqMax初期設定」で環境を最適化します。
4. 「02. 各連携設定」で利用したい通知先を設定し、ボットを配置します。


--------------------------------------------------
【実行ステップ：ALL GREEN への道】
--------------------------------------------------
1. ボットを実行すると、Step 1/7 ～ 7/7 の診断シーケンスが開始されます。
2. ライブラリ、環境パス、Webhookの状態を一つずつチェックします。
3. 最後に【SYSTEM STATUS: ALL GREEN】と表示されれば監視開始です。
4. 監視中、約1時間ごとの生存報告は「ReportEvery 3600s」として出力されます。


--------------------------------------------------
【重要：起動トラブルと「セーフモード」について】
--------------------------------------------------
ボットが起動しない等の場合は【セーフモード診断】を行ってください。
1. 配置ツールで作成された「(セーフモード)」ショートカットを実行してください。
2. 自動的にPython環境の整合性チェックと、不足ライブラリの修復が始まります。
