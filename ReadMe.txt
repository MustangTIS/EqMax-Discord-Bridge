==================================================
EqMax-Discord-Bridge Project (v11.0.0 Major Update)
==================================================

この度は本ツールをご利用いただきありがとうございます。
本パッケージは、強震モニタ表示ソフト「EqMax」の運用を最適化し、
地震速報を Discord, Bluesky, Slack, Matrix 等の複数プラットフォームへ即時通知するための
統合型防災通知ハブです。

【v11.0.0：The Blue Horizon - Bluesky Integration】

[最新アップデート：v11.0.0]
● Bluesky への正式対応：
  X(旧Twitter)のAPI有償化に伴い、新たな発信先としてBluesky投稿機能を実装。
  地震速報を迅速に分散型SNSのタイムラインへ届けます。
● 堅牢な自動判別ブーターの導入：
  実行環境のPythonパスを自動解析する新ロジックを搭載。
  「python」または「py」コマンドの有無を自動判別し、環境構築のトラブルを最小限に抑えます。
● 確定震度情報・津波情報の統合：
  EEW（速報）に加え、詳細な「震源・震度報」および「津波警報・注意報」の自動通知を完備。
● ニュース速報スタイルフォーマット：
  確定報の表示順を『震度 ＞ 地域 ＞ 市区町村』の順に構成。
  テレビのニュース速報と同様の感覚で、被害の大きい地域を即座に判別可能です。
● Discord Embed 拡張対応：
  情報の最大出力を、従来の2000文字からDiscord Embedの上限である4000文字まで拡張しました。

【動作環境・スペック】
開発＆動作確認: Windows 11 / Windows Server 2025
動作環境: Windows 10/11, Windows Server 2022/2025 (x64専用)
必要ソフト: Discord, EqMax, Python (3.10-3.14対応) 

【同梱ツールの紹介】
● [HTMLマニュアル/ヘルプ]
  導入からBluesky連携手順までを網羅した完全運用ガイド。
● [統合ハブツール (TOP_HUB)]
  2カラムレイアウトに刷新された、全機能を呼び出すメインメニュー。
● [通知ボット (Guardian) / Watchdog]
  メモリ監視およびフリーズ検知機能を備えた24時間監視エンジン。
● [EEW/情報強化テスト送信ユニット]
  確定報・津波情報を含めた通知テストが可能な診断ツール。

【導入ステップ】
1. 「EqMax-Discord-Bridge.bat」を実行し、統合ハブを起動します。
2. 画面上のボタンから「HTMLヘルプ」を参照し、各設定手順を確認します。
3. 「01. EqMax初期設定」で環境を最適化します 。
4. 「02. 各連携設定」で利用したい通知先を設定し、ボットを配置します。
5. 「テスト投稿」を実行し、正常に通知が届くことを確認してください。
