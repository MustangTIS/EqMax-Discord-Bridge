==================================================
  EqMax-Discord-Bridge Project (v6.0.0 Emergency Update)
==================================================

この度は本ツールをご利用いただきありがとうございます。
このパッケージは、強震モニタ表示ソフト「EqMax」の運用と
Discord通知を劇的に便利にするための統合ツール群です。

【重要】v6.0.0：Twitter(X)通信の完全遮断による安定性の確保
近時のCloudflareによるIPブロック問題（Sorry, you have been blocked）への緊急対策として、
EqMax内部の通信フラグ TwitterDummyPost=1 を活用した「通信完全分離モード」を標準搭載しました。
これにより、API通信に起因するブラウザのフリーズやネットワーク制限を物理的に回避し、
Discord通知に特化した究極の安定稼働を実現します。

【同梱ツールの紹介】
--------------------------------------------------
● [統合管理ハブ]
   全ての機能を呼び出すメインメニューです。まずはこれを実行してください。

● [Eqmax初期設定ツール (Patcher)]
   EqMaxの面倒な初期設定を一括で行い、動作を軽量化・安定化させます。
   v6.0.0では、ボタン一つで「Twitter通信の安全な遮断」と「疑似認証の構築」を自動で行います。

● [Discord設定 (Guardian)] / [Watchdog (単体版)]
   v5.6.0より、物理メモリ（RSS）では検知しきれなかった「予約領域(Private Bytes)」まで
   厳密に監視する新エンジンに刷新されました。
   長期間の無人運用におけるメモリリーク対策が極限まで強化されています。

● [EqMAX初期化ツール]
   カスタマイズやログ、Discord設定を含め全て削除し、導入直後の状態に戻します。

● [オプション：ログ・画像掃除]
   肥大化しやすいTwitter.logやキャプチャ画像を清掃します。

--------------------------------------------------
【導入ステップ】
--------------------------------------------------
1. 同梱の「EqMax-Discord-Bridge.bat」を実行します。
2. 「00-TOP_HUB」が立ち上がったら、まず「01. EqMax初期設定」で最適化を行います。
3. 次に「02. Discord連携」でWebhookや監視パラメータを設定し、ボットを配置します。
   ※配置ツールでは「通常」と「セーフモード」のショートカット作成を選択可能です。

--------------------------------------------------
【実行ステップ：ALL GREEN への道】
--------------------------------------------------
1. ボットを実行すると、Step 1/7 ～ 7/7 の診断シーケンスが開始されます。
2. ライブラリ、アップデート、環境パス、Webhookの状態を一つずつチェックします。
3. 最後に【SYSTEM STATUS: ALL GREEN】と表示されれば、監視開始です。
4. 監視中、約1時間ごとの生存報告は「ReportEvery 3600s」として出力されます。

--------------------------------------------------
【重要：起動トラブルと「セーフモード」について】
--------------------------------------------------
もし「ボットが起動しない」「黒い画面がすぐ閉じる」といった現象が起きた場合は、
以下の手順で【セーフモード診断】を行ってください。

1. 配置ツールで作成された「(セーフモード)」ショートカットを実行するか、
   各フォルダ内にあるバッチファイル（.bat）を直接実行してください。
2. 自動的にPython環境の整合性チェックと、不足ライブラリの修復が始まります。
3. 画面に「ATTENTION REQUIRED」と出た場合は、表示された指示に従ってください。
