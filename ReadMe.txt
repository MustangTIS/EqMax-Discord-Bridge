==================================================
  EqMax-Discord-Bridge Project (v5.5.3)
==================================================

この度は本ツールをご利用いただきありがとうございます。
このパッケージは、強震モニタ表示ソフト「EqMax」の運用と
Discord通知を劇的に便利にするための統合ツール群です。

--------------------------------------------------
【動作環境・スペック】
--------------------------------------------------
開発＆動作確認
Windows 11 / Windows Server 2025

動作環境
Windows 10/11, Windows Server 2022/2025 (x64専用)
※ARM版Windowsではエミュレーション動作は可能と思われますが、動作としてはネイティブx64を推奨します。

必要ソフト
Discord https://discord.com
Eqmax https://melanion.info/eqmax/
Python(3.10-3.13推奨）https://www.python.org/downloads/windows/

--------------------------------------------------
【同梱ツールの紹介】
--------------------------------------------------
● [統合管理ハブ]
   ここから全ての操作を開始します。
   全ての機能を呼び出すメインメニューです。まずはこれを実行してください。

● [Eqmax初期設定ツール] (初期設定)
   EqMaxの面倒な初期設定を一括で行い、動作を軽量化・安定化させます。
   「Full構成（通知優先）」と「Server構成（負荷軽減）」を選択可能です。

● [Discord設定] (EqmaxのEEWをWebhook経由でDiscordに送信するソフト)
   Webhook URLを登録するだけで連携が完了します。
   リッチな「埋め込み(Embed)」と、1行で済む「シンプル」から選べます。

● [EqMAX初期化ツール] (全ての設定を初期化しソフトウエア導入時に戻す)
   EqMAXのカスタマイズやログ及びDiscord設定など含め全て削除し初期設定に戻します。
   DL、解凍、再配置無しで始めれるので便利です。

● [オプション：ログ・画像掃除] (EqMAXのクリーニングソフト)
   溜まったログや画像を清掃します。EqMaxとの競合を防ぐ自動終了機能を搭載。
   Twitter.logやキャプチャフォルダ・Twitterフォルダなどは非常にゴミをため込むので定期的にクリーニングすることをオススメします。

● [オプション：Watchdog]
   メモリ超過やフリーズを監視し、自動復旧させます。
   ※Discord連携ツールにはこの機能が標準で内蔵されています。

--------------------------------------------------
【導入ステップ】
--------------------------------------------------
1. 同梱の「EqMax-Discord-Bridge.bat」を実行します。
2. 「00-TOP_HUB」が立ち上がったら、まず「01. EqMax初期化」で最適化。
3. 次に「02. Discord連携」でWebhookを設定。
4. 「監視ボットの起動」を押せば、通知と死活監視がスタートします。
