==================================================
  EqMax-Discord-Bridge Project (v5.6.0)
==================================================

この度は本ツールをご利用いただきありがとうございます。
このパッケージは、強震モニタ表示ソフト「EqMax」の運用と
Discord通知を劇的に便利にするための統合ツール群です。

v5.6.0より、監視エンジンの刷新と「起動診断シーケンス」を導入しました。

--------------------------------------------------
【動作環境・スペック】
--------------------------------------------------
開発＆動作確認
Windows 11 / Windows Server 2025

動作環境
Windows 10/11, Windows Server 2022/2025 (x64専用)
※ネイティブx64環境を推奨します。

必要ソフト
- Discord https://discord.com
- Eqmax https://melanion.info/eqmax/
- Python (3.10-3.13推奨) https://www.python.org/downloads/windows/

--------------------------------------------------
【同梱ツールの紹介】
--------------------------------------------------
● [統合管理ハブ]
   全ての機能を呼び出すメインメニューです。まずはこれを実行してください。

● [Eqmax初期設定ツール]
   EqMaxの面倒な初期設定を一括で行い、動作を軽量化・安定化させます。
   「Full構成（通知優先）」と「Server構成（負荷軽減）」を選択可能です。

● [Discord設定 (Guardian)]
   Webhook経由でDiscordに送信し、同時にEqMaxのフリーズやメモリを監視します。
   v5.6.0より、物理メモリだけでなく「予約領域(Private Bytes)」まで厳密に監視する
   新エンジンに刷新されました。

● [EqMAX初期化ツール]
   カスタマイズやログ、Discord設定を含め全て削除し、導入直後の状態に戻します。

● [オプション：ログ・画像掃除]
   肥大化しやすいTwitter.logやキャプチャ画像を清掃します。

● [オプション：Watchdog (見張り番)]
   EqMaxの生存確認に特化した単体動作版です。Discord連携が不要で、
   純粋にEqMaxのフリーズ・メモリ監視のみを行いたい場合に最適です。
   ※Discord連携ツール(Guardian)には、この機能が標準で統合されています。


--------------------------------------------------
【導入ステップ】
--------------------------------------------------
1. 同梱の「EqMax-Discord-Bridge.bat」を実行します。
2. 「00-TOP_HUB」が立ち上げたら、まず「01. EqMax初期化」で最適化。
3. 次に「02. Discord連携」でWebhookを設定。
4．導入が完了したら、回答した本ソフトウエアは削除したとしても動作します。
   メンテナンスソフトなども便利なのですが、再度DLしてもいいですし、使い方はご自由に。

--------------------------------------------------
【実行ステップ】
--------------------------------------------------
1. EqMax-Discord通知ボット(Watchdogアプリの場合はEqMax-Watchdog)を実行するとコンソールが起動します。
2. アプリが起動すると最小化されたコンソールが起動します。
3. 最後に【SYSTEM STATUS: ALL GREEN】と表示されれば、監視開始です。

--------------------------------------------------
【重要：起動トラブルと「セーフモード」について】
--------------------------------------------------
もし「ボットが起動しない」「黒い画面がすぐ閉じる」といった現象が起きた場合は、
以下の手順で【セーフモード診断】を行ってください。

1. 各ツールのフォルダ内にあるバッチファイル（.bat）を直接ダブルクリックで実行。
2. 自動的にPython環境やライブラリの整合性チェックが始まります。
3. 不足しているライブラリがあれば自動修復を試み、修復後は通常通り起動します。

OSクリーンインストール後や、Pythonの誤削除、別PCへの環境移行直後などは、
この「バッチファイルからの直接起動」を一度行うことを強く推奨します。
