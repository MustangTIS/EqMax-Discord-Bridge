==================================================
EqMax-Discord-Bridge Project (v9.0.0 Update)
==================================================

この度は本ツールをご利用いただきありがとうございます。
本パッケージは、強震モニタ表示ソフト「EqMax」の運用を最適化し、
地震速報を Discord, Slack, Matrix 等の複数プラットフォームへ即時通知するための
統合型防災通知ハブです。

【v9.0.0：Major UI & Functional Update】

[最新アップデート：v9.0.0]

総合管理ハブのUIを刷新し、2列構成のモダンなレイアウトへ変更。

「EEWテスト投稿ツール」を新規搭載。実際の地震を待たずに通知テストが可能に。

Discord連携設定ツールのUIをスクロール対応にし、利便性を向上。

設定更新時にバックグラウンドで動作中の古いプロセスを自動停止する機能を実装。

マニュアル（HTML）を最新のツール構成に合わせて大幅にリライト。

[v8.6.0]

通知ボット配置ツールに「既存のSNS/通知先設定の読み込み機能」を追加。
Webhookやトークンの再入力の手間を大幅に削減しました。

【動作環境・スペック】

開発＆動作確認: Windows 11 / Windows Server 2025
動作環境: Windows 10/11, Windows Server 2022/2025 (x64専用) ※ネイティブx64環境を推奨。
必要ソフト: Discord, Eqmax, Python (3.10-3.13推奨)

【同梱ツールの紹介】

● [HTMLマニュアル/ヘルプ]
アプリ内の全機能・設定方法・対応表を網羅した完全運用ガイド。

● [統合ハブツール (TOP_HUB)]
全てのメンテナンス・設定機能を呼び出すメインメニューです。

● [Eqmax初期設定ツール (Patcher)]
EqMaxの面倒な初期設定を一括で行い、動作を軽量化・安定化させます。

● [通知ボット (Guardian) / Watchdog]
メモリ(Private Bytes)監視およびフリーズ検知機能を備えた高安定監視エンジン。

● [EEWテスト投稿ツール]
現在の設定で正しく通知が飛ぶかを検証するための診断ツール。

● [ログ・画像掃除ツール (Cleaner)]
肥大化しやすいログやキャプチャ画像を清掃します。

● [EqMAX初期化ツール (Reset)]
すべての設定を消去し、インストール直後の状態に戻します。

【導入ステップ】

「EqMax-Discord-Bridge.bat」を実行し、統合ハブメニューを起動します。

画面上のボタンから「HTMLヘルプ」を参照し、各設定手順を確認します。

「01. EqMax初期設定」で環境を最適化します。

「02. 各連携設定」で利用したい通知先を設定し、ボットを配置します。

「EEWテスト投稿」を実行し、正常に通知が届くことを確認してください。

【重要：起動トラブルと「セーフモード」について】

ボットが起動しない等の場合は【セーフモード診断】を行ってください。

配置ツールで作成された「(セーフモード)」ショートカットを実行してください。

自動的にPython環境の整合性チェックと、不足ライブラリの修復が始まります。
