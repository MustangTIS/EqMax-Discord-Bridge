更新履歴 (ChangeLog)

EqMax-Discord-Bridge のこれまでのアップデート内容です。

v7.1.0 (Integration & Stability Update)

「境界を超えた情報の、その先へ」 — 接続性の拡張と、視認性の最適化。

  • [Protocol] Matrix API v3 完全対応: 最新サーバー環境での安定性を向上。
  • [Visual] 視認性の向上: Slack/Matrix向けに「通知区切り線」と「送信時刻(timestamp)」を自動付与。
  • [Quality] GUIインストーラの堅牢化: Webhook検証強化、クリップボード貼り付け対応。

通知イメージの刷新:

📢 [EqMax] EEW Alert📢 【最大震度 2 / M4.5】 / 送信時 14:27:02 / ...
───────────
📢 [EqMax] EEW Alert📢 【最大震度 2 / M4.6】 / 送信時 14:27:06 / ...

v7.0.0 (マルチプラットフォーム対応)

「境界を超えて、全方位へ」 — マルチプラットフォーム通知の夜明け。

  • [Platform] マルチ通知エンジン「Guardian Hub」始動: Discord, Slack, Matrix への同時通知に対応 [cite: 1, 2]。
  • [Identity] Webhook送信名の強制統合: 送信名を EqMax EEW Bridge に統一。
  • [Stability] エラー診断プロンプトの強化: 送信失敗時の原因特定を容易に。

v6.0.0 (Emergency & Performance Update)

「静かなる守護者」 — 通信の完全分離と、究極の安定性へ。

  • [Network] Twitter(X)機能の完全デッドロック回避: CloudflareによるIPブロックやフリーズを根絶。
  • [Setup] 「ワンクリック・パッチ」の実装: 初期設定を全自動化。

v5.0 - v5.6.x (Stability & Interface)

  • [Core] メモリリーク対策: Private Bytes監視への移行により安定性を向上。
  • [Safety] セーフモード診断: 自己診断・自動修復機能の実装 [cite: 3]。
  • [UX] ブランドデザイン統合: 公式ロゴ表示とハブメニューの再設計。

v1.0 - v4.5 (Foundation)

  • Watchdog機能による24時間365日の自動監視・再起動ロジック搭載 [cite: 3, 4]。
  • ログやキャプチャ画像の一括清掃機能の実装 [cite: 4]。
