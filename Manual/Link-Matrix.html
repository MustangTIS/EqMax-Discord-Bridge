Matrix連携設定

総合ハブの Discord連携設定 (➃の部分)を実行してください。

1. Discord連携ツールの起動と選択

  1. EqMax.exeの選択: 連携したいEqMax本体を選択します。
  2. 連携先タイプ: 「Matrix」を選択してください。

2. 各情報の取得方法

ホームサーバーURL

ログイン時に使用するアドレスです。これをツール側の ホームサーバーURL 欄に入力してください。

ルームID

  1. ルーム右上の「ギアマーク」＞「設定」を開きます。
  2. 「詳細」タブ内の「ルームID」をコピーし、ツール側の Room ID 欄に入力してください。

アクセストークン

  1. 左下の「ギアマーク」＞「すべて設定」を開きます。
  2. 「ヘルプ」＞「アクセストークン」を選択し表示されたTokenをコピーします。
  3. ツール側の Access Token 欄に入力してください。

※警告: アクセストークンは乗っ取りのリスクがあります。決して他人に教えないでください。

3. ツール側への入力・保存

取得した情報を、Discord連携ツール（アプリ）の以下の各欄に貼り付け、「保存」ボタンを押してください。

  • ホームサーバーURL: 手順2-1で取得したURL
  • Room ID: 手順2-2で取得したRoom ID
  • Access Token: 手順2-3で取得したToken

設定完了後、一度テスト通知を行い、正しくメッセージが届くか確認してください。
