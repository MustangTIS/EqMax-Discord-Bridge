EqMax-Discord-Bridge

～EqMaxの緊急地震速報をDiscord・Slack・Matrixへ通知するためのセットアップキット～

はじめに

このたびは、緊急地震速報通知ツール『EqMax-Discord-Bridge』をご利用いただきありがとうございます。

本プロジェクトは、地震情報表示ソフト「EqMax」が受信する高度な速報データを、Discord、Slack、Matrixといった主要なコミュニケーションツールへ即座に、かつ確実に橋渡し（Bridge）することを目的に開発されました。

本ツールの特徴とメリット

  • 完全ローカル・セルフホスト運用 一般的な外部連携サービスに依存せず、ご自身のPCや自宅サーバーを拠点として動作します。外部サービスのダウンや遅延に左右されず、自分専用の堅牢な通知環境を構築できます。

  • マルチプラットフォーム対応 Discordの多彩な埋め込み表示（Embed）はもちろん、ビジネスユースのSlackや、分散型プロトコルのMatrixまで、複数の宛先へ同時に情報を配信可能です。

  • 無限のカスタマイズ性 本ツールはオープンソースで公開されています。Pythonベースの柔軟な設計により、スマートスピーカーによる音声読み上げや、スマート照明との連動など、あなたのアイデア次第で「次世代の防災システム」へと拡張することが可能です。

  • 実用性を極めたフォーマット v10.0.0からは、EEWとは別に地震情報をテレビのニュース速報のような「震度順」表示を採用。緊急時に最も重要な「どこが一番揺れているか」を直感的に把握できる、現場主義のインターフェースを実現しました。

マニュアルの構成

各セッションはサイドバーから選択して移動できます。

動作環境・開発環境

  • 開発＆動作確認: Windows 11 / Windows Server 2025
  • 対応OS: Windows 10/11, Windows Server 2022/2025 (x64専用) ※ネイティブx64環境を推奨。

必要環境ソフト

  • Python: 公式サイト
