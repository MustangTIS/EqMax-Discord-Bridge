## ## EqMax Discord連携ツール 導入ガイド

### ■ システム概要

本システムは、EqMax の地震情報を Discord へリアルタイム転送するためのパッケージです。
最大の特徴として、**最大5箇所の Discord Webhook を同時登録可能** であり、用途に合わせて「リッチな埋め込み表示」と「スマートな1行表示」を使い分けられます。

---

### ■ ステップ1：初期設定ツール (01-EqMax初期設定)

EqMax 本体を Discord 連携に最適化された状態へ自動的に構成します。

* **主な機能**:
* `EqMax.ini` の通知フォーマットや画像保存設定を自動更新します。
* 連携に必要なログファイル（Twitter.log）を自動生成します。
* 実行時、EqMax が起動していれば自動で安全に終了（強制終了）させてから処理を行います 。




* **使い方**:
1. 初回のみ `PythonINS\install_lib.bat` を実行し、必要な部品を導入します。
2. 
`start_app.bat` を実行し、EqMax.exe を選択してセットアップを開始してください 。





---

### ■ ステップ2：Discord連携ツール (02-Discord連携)

通知を Discord へ飛ばすための「監視ボット」を配置します。

* **主な機能**:
* **マルチ Webhook 対応**: 最大5件までの Webhook URL を登録でき、それぞれに異なる表示スタイルを設定可能です。
* 監視環境の自動構築を行い、EqMax フォルダ内に運用専用の `Discordbot` フォルダを作成します。


* **使い方**:
1. `start_app.bat` を実行し、画面の案内に従って Webhook URL と表示スタイル（Embed / Simple）を入力してください。



**【スタイルの選び方】**

* **Embed (推奨)**: Discord で画像付きのリッチな通知を行いたい場合に最適です。
* **Simple**: 1行に情報を集約します。**Discord 以外のツール（Slack 等）へ Webhook を送信する場合**や、スマホ・スマートウォッチの通知欄で内容を即座に確認したい場合に推奨されます。

---

### ■ ステップ3：おまけ・ログクリーナー (03-EqMAX内ログクリーナー)

運用で溜まったデータ（画像・ログ）を自動で掃除し、PC を軽く保ちます。

* **主な機能**:
* 指定した日数（保持期間）を過ぎたキャプチャ画像や過去ログを一括削除します。
* ステップ1と同様、実行時に EqMax を自動で終了させてから安全に処理を行います。



---

### ■ 起動と運用

* **自動起動設定**: セットアップ時にスタートアップ登録を行うと、PC 起動時に自動で監視が開始されます。
* **手動起動**: 生成されたショートカット、または `Discordbot\eqmax_discord.bat` から起動してください。

---

### ■ 導入時の注意

* **Pythonの準備**: インストール時は必ず **「Add Python to PATH」** にチェックを入れてください。
* **自動復旧**: 各ツールの起動に失敗した場合、バッチファイルが自動的に不足ライブラリの修復を試みます。
