### ■ 導入手順

#### **【重要：Windowsのセキュリティ警告について】**

本ツールは個人開発の未署名ファイルであるため、ダウンロード後や初回実行時に Windows SmartScreen（青い画面）が表示されることがあります。 

* 
**対処法**: 「詳細情報」をクリックし、「実行」を選択してください。 


* **ブロック解除のお願い**:
1. ダウンロードした ZIP ファイルを右クリックして「プロパティ」を開く。
2. 全般タブの一番下にある **「許可する」または「ブロックの解除」** にチェックを入れて OK を押す。


* これを行うことで、セットアップ後のスタートアップ起動や自動復旧機能（見張り番）が、警告で止まることなくスムーズに動作するようになります。

#### 1. 準備：環境構築

ルートディレクトリにある **`EqMax-Discord-Bridge.bat`** を実行してください。

* ライブラリが不足している場合、自動的にセットアップが開始されます。

#### 2. メインハブ (TOP_HUB) の操作

ハブ画面から、以下の順に設定を進めてください。

### ■ 各ツールの役割

#### ① 初期設定パッチ (01-Eq_Initialize)

EqMaxを「Discord連携モード」へ自動構成します。

* `EqMax.ini` の書き換え、ログファイルの自動生成、起動中の EqMax の安全な終了を行います。

#### ② ボット配置ツール (02-Eq_Discord)

通知システムを EqMax フォルダ内へインストールします。

* **最大5件の Webhook** を同時登録可能。
* **Embed**: 画像付きのリッチな通知（Discord特化）。
* **Simple**: 1行のスマートな通知。Discord以外のシステム（Slack、LINE、Matrix等）にオススメです。
* **[NEW]** v4.0より、プロセスのフリーズ監視やメモリリーク対策の「見張り番」機能がボット本体に統合されました。

#### ③ 【重要】初期化・修復ツール (03-Eq_Reset)

**「設定ミス」や「動作不良」時の最終手段です。**

* 配置したボットファイルや、書き換えた設定を削除し、EqMaxをインストール直後の状態へ戻します。

#### ④ ログ・画像クリーナー (O01-Eq_Cleaner)

日常的なメンテナンスツールです。

* 溜まり続けるキャプチャ画像や肥大化したログを安全に削除し、PCを軽く保ちます。

#### ⑤ 動作監視スクリプト (O02-Eq_Watch)

長期間の無人運用をサポートするウォッチドッグです。

* ※この機能は **「② Discord連携ツール」に既に実装済み** です。通常、ボットを導入している環境では併用の必要はありません（二重監視による競合を防ぐため非推奨）。

### ■ 運用開始

セットアップ完了後は、デスクトップの **「EqMax-Discord通知ボット」** ショートカットを実行してください。
