# EqMax-Discord-Bridge
EqMaxの緊急地震速報をDiscordへ自動転送・画像付きで通知するためのセットアップキット（INI自動パッチ・保守ツール同梱）


# EqMax Discord連携ツール 導入ガイド

## 0. 事前準備

1. EqMax を公式サイトからダウンロードしてください。  
   https://melanion.info/eqmax/

2. 展開先は「C:\Program Files」などの保護フォルダを避け、  
   例：`C:\EqMax`  
   書き込み権限のある場所に配置してください。  
   ※保護フォルダではパッチが正常に動作しない場合があります。

---

## 1. EqMax 内部設定（ステップ1）

1. `01-EqMax初期設定` フォルダを開きます。
2. 初回のみ `PythonINS\install_lib.bat` を実行し、必要ライブラリをインストールします。
3. `start_app.bat` を実行し、`EqMax.exe` を選択してセットアップします。

実行後、`EqMax.ini` のキャプチャ設定やフォーマットが自動更新されます。  
また、連携用の `Twitter.log`（0KB）などが自動生成されます。

---

## 2. Discord 連携配置（ステップ2）

1. `02-Discord連携` フォルダ内の `start_app.bat` を実行します。
2. 画面の案内に従い、以下を入力します。
   - EqMax フォルダの選択
   - Discord Webhook URL
   - （任意）ショートカット作成、スタートアップ登録

3. 完了すると、EqMax フォルダ直下に `Discordbot` フォルダが作成され、必要ファイルがコピーされます。

---

## 3. 起動と運用

- 同時起動  
  作成されたショートカット、または `Discordbot\eqmax_discord.bat` を実行すると、  
  EqMax 本体と監視ボットが同時に起動します。

- 自動起動  
  スタートアップ登録を有効にした場合、PC 起動時に自動で監視が開始されます。

---

## 4. メンテナンス（ステップ3・おまけ）

1. `03-EqMAX内ログクリーナー-おまけ` フォルダを開きます。
2. `start_app.bat` を実行し、`EqMax.exe` を選択します。
3. 削除したいログや画像の「保持日数」を選択して実行します。

長期間運用して動作が重くなった場合や、ディスク容量を節約したい場合に有効です。

---

## 注意事項

- セットアップやクリーナー実行時は、必ず **EqMax 本体を終了** させてください。
- 初回実行でエラーが出る場合は、各フォルダ内の `install_lib.bat` を先に実行してください。
- Python をインストールする際は、**「Add Python to PATH」** に必ずチェックを入れてください。
